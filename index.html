강력하다

Alpine은 마크업에 직접 작성할 수 있는 도구입니다. 최신 웹용 jQuery라고 생각하면 됩니다. 스크립트 태그를 넣고 시작하세요.

Alpine.js는 15개의 속성, 6개의 프로퍼티 그리고 2개의 메서드로 구성된 컬렉션입니다.

직접 눈으로 확인해 가면서 배우는 것이 가장 좋은 방법입니다.

15개의 속성

  • x-data

    HTML 블록에 새 Alpine 컴포넌트와 데이터를 선언한다. x-로 시작한다.

  • x-bind

    요소에 HTML 속성을 동적으로 설정하기. div 에 클래스 속성을 추가하거나 제거할 수 있다.

  • x-on

    엘리먼트에 브라우저 이벤트 수신

  • x-text

    엘리먼트에 텍스트 콘텐츠 설정

  • x-html

    엘리먼트에 내부 HTML 설정

  • x-model

    input 요소와 데이터를 동기화하기

  • x-show

    엘리먼트의 표시 여부를 토글하기 (on/off)

  • x-transition

    CSS 전환을 사용하여 엘리먼트 안팎으로 전환하기

  • x-for

    데이터 집합을 기반으로 HTML 블록을 반복한다.

  • x-if

    페이지에서 HTML 블록을 조건부로 추가/제거한다.

  • x-init

    알파인에 의해 엘리먼트가 초기화될 때 코드 실행

  • x-effect

    종속성 중 하나가 변경될 때마다 스크립트가 실행된다.

  • x-ref

    $ref 매직 속성을 사용해서 지정된 키로 요소를 직접 참조한다.

  • x-cloak

    알파인이 콘텐츠 초기화를 완료할 때까지 HTML 블록을 숨긴다.

  • x-ignore

    알파인이 HTML 블록을 초기화하지 못하도록 방지합니다.

6개의 프로퍼티

  • $store

    Alpine.store(...) 을 이용해서 등록된 글로벌 스토어에 액세스합니다.

  • $el

    현재 DOM 요소 참조

  • $dispatch

    현재 요소에서 사용자 지정 브라우저 이벤트 디스패치하기

  • $watch

    데이터를 감시하고 변경될 때마다 제공된 콜백을 실행한다.

  • $refs

    키로 요소 참조(x-ref를 사용하여 지정)

  • $nextTick

    다음 "tick" (브라우저 페인트)이 실행될 때까지 기다렸다가 약간의 코드를 실행한다.

2개의 메서드

  • $store

    Alpine.store(...) 을 이용해서 등록된 글로벌 스토어에 액세스합니다.

  • Alpine.data

    데이터 객체를 재사용하고 x-data를 사용하여 참조하기

  • Alpine.store

    $store를 사용하여 어디서나 액세스 할 수 있는 글로벌 반응형 데이터를 선언한다.
